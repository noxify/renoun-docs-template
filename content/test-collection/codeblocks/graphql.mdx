---
title: GraphQL Grammar
---

## graphql

```graphql
query {
  endpoint(sorting: {field: "fieldName", direction: ASC}) {
    nodes {
    fieldname1
      alias: fieldName2
    }
  }
}
```

## gql

```gql
query {
  endpoint(sorting: {field: "fieldName", direction: DESC}) {
    nodes {
    fieldname1
      alias: fieldName2
    }
  }
}
```
